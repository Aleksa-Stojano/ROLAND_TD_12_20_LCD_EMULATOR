# Contributing

Thank you for your interest in this project.

## Getting started

1. Clone the repository.
2. Install build dependencies (see [docs/usage.md](docs/usage.md)).
3. Run `make` to build `td_12_lcd_emulator`.

## Documentation

- [Usage guide](docs/usage.md): build, capture, run, troubleshoot.
- [Background](docs/background.md): motivation and reverse-engineering notes.
- [Architecture](docs/architecture.md): decoder design and file layout.

Please update the relevant doc when you change behaviour, wiring assumptions, or the capture pipeline.

## Pull requests

1. Keep changes focused on one topic where possible.
2. Ensure the project still builds with `make`.
3. Describe what you changed and why in the pull request.
